Wireframer Help

Workflow

Drag primitives from the left panel onto the canvas or into containers.
Click any node to select it and edit its properties in the Inspector.
Nest nodes by dropping onto a Container (dashed border turns green).
Use X-Ray to reveal all container boundaries.

Shortcuts

Ctrl+SSave design to Fragment KV
Ctrl+ZUndo
Ctrl+Shift+ZRedo
Ctrl+DDuplicate selected node

Node Search

Type in the search bar and press Enter to find nodes by alias or type. Matching nodes highlight in amber; non-matching nodes dim. Press Enter again to cycle through results. ✕ or Escape clears.

Import / Export

Import — paste or upload a .yaml or .html file to replace the canvas.
Export YAML — downloads the raw tree as a YAML file. Re-importable.
Export HTML — generates semantic HTML5 at the current frame width.
→ Fragment — pushes exported HTML directly to your Fragment KV library (requires unlock).

Inspector Sections

Click any section header to collapse or expand it. Useful on small screens.

Node Types

▢ ContainerFlexbox wrapper. Only type that holds children.
T Texth1–h3, p, span, label. Set tag for semantic HTML.
▣ ButtonPrimary, secondary, or disabled weight.
▭ InputText, email, password, number, search, tel.
