# Input & Controls Reference

## Keyboard (Desktop)

### Movement
- W: Move up
- A: Move left
- S: Move down
- D: Move right

### Combat
- Mouse LMB (click): Fire weapon
- R: Reload
- F: Throw grenade
- Q: Swap primary/secondary weapon
- Mouse wheel up/down: Weapon swap (alternative)

### UI
- Escape: Pause/open menu
- (Click buttons in shop/harvest/pause screens)

### Settings
- Tab: Toggle dev panel (if enabled in settings)
- (Toggle buttons: virtual gamepad, screen shake, dev overrides)

## Virtual Gamepad (Mobile)

### Left Side (Movement)
- Joystick drag: Directional movement (8-way)
- Touch and drag in left half of screen to move joystick

### Right Side (Actions)
- FIRE (large red circle): Shoot
- DASH (blue): Dodge/dash
- BOMB (orange): Throw grenade
- SWAP (cyan): Swap weapons

## Menu Navigation

### Boot Screen
- DEPLOY button: Start game

### Pause Menu
- RESUME: Back to game
- SETTINGS: Open settings panel

### Settings Panel
- Toggle buttons: Virtual Gamepad, Screen Shake, Dev Overrides
- CLOSE: Return to game (or boot if not in game)

### Shop Screen
- Click weapon in rack to select
- Click track buttons to upgrade (if affordable)
- Click EQUIP PRIMARY/SECONDARY to equip
- Click tabs: FIREARMS, LOGISTICS, VENTURE
- RETURN TO BATTLEFIELD: Go to next wave

### Harvest Screen
- Click mutation card to select (choose 1 of 3)

## Dev Panel (If Enabled)

- GOD MODE: Toggle invincibility
- INJECT FUNDS ($10k): Spawn money
- UNLOCK ALL WPN: Unlock all weapons
- KILL ALL (SKIP WAVE): Force wave clear
- RESTART WAVE: Go back one wave

## Notes
- Mobile auto-detects and shows virtual gamepad if enabled in settings
- No mouse aim on mobile; fire direction based on movement
- Keyboard + mouse assumed for desktop
- Can change keys by editing game.js input handlers (future feature)
